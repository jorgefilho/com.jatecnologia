Como remover erros de validacao Javascript no jquery minified:
- Project -> Properties -> Javascript -> Include Path -> Source -> ns-rma-fe/src/main/webapp -> Excluded -> Edit... -> Exclusion patterns -> Add...
- Na caixa de texto, digitar: js/jquery-*.min.js
- OK -> Finish -> OK
- (fonte: http://stackoverflow.com/questions/3131878/how-do-i-remove-javascript-validation-from-my-eclipse-project)
