A)Settings - top

Set up Latest Tweets

If you don't want to set up twitter please comment twitter function from custom.js

open file 

 js/custom.js
Find 
username: "pxcreate",
replace it with your username like, 
username: "xyz",
Find 
// modpath: 'catalog/view/theme/shopsimple/twitter/',
replace it with Below (or just remove comment) 
modpath: 'catalog/view/theme/shopsimple/twitter/',

& for Twitter Feed API 1.1 support and OAuth Token Authentication integration

open file 

 twitter/index.php
Find 
// Your Twitter App Consumer Key
	private $consumer_key = 'YOUR_CONSUMER_KEY';

	// Your Twitter App Consumer Secret
	private $consumer_secret = 'YOUR_CONSUMER_SECRET';

	// Your Twitter App Access Token
	private $user_token = 'YOUR_ACCESS_TOKEN';

	// Your Twitter App Access Token Secret
	private $user_secret = 'YOUR_ACCESS_TOKEN_SECRET';
replace it with yours. You can get all detials from here, https://dev.twitter.com/apps Login with your twitter ID & create Application.
You need to have a twitter App for your usage in order to obtain OAuth credentials.

C) JavaScript - top

This theme include below files.

  1. Revolution Slider all js, color variation & Documentation
  2. html5.js - should be on top
  3. respond.min.js -For css Media query in IE8
  4. zoomsl-3.0.min.js - For Zoom plugin
  5. socialstream.jquery.js - For Social Feed
  6. jquery.flexslider.js -For Responsive Slider & carousal
  7. jquery.prettyPhoto.js -For Popup Plugin
  8. carouFredSel-6.1.0-packed.js -For Responsive Slider & carousal
  9. jquery.mousewheel.min.js - To make slide & carousal work with mousewheel
  10. jquery.jquery.touchSwipe.min.js - To make slide & carousal work with TouchSwipe
  11. jquery.validate.js - For Form Validation
  12. jquery.gmap.js - for google map
  13. jquery.isotope.min.js - for Animated Portfolio
  14. jquery.countdown.js - for Coming Soon page Contdown
  15. custom.js - all setting can be made from here
